Sumá **vendedores** (o canales) con su propio **link de referido**: cada uno comparte su enlace y vos ves cuánto vendió.

## Crear un link de vendedor

Para un evento, creás un **link** con una **etiqueta** (el nombre del vendedor o canal) y, si querés, un **código** propio (si no, se genera uno). Podés asociarlo a un **integrante de tu equipo**.

A cada link le definís una **recompensa** (su comisión): un **valor** con su **tipo**, para saber cuánto le corresponde por lo que vende.

## Seguimiento

Cada link acumula automáticamente:

- **Clicks** — cuánta gente entró por ese enlace.
- **Entradas vendidas** e **ingresos atribuidos** — la venta se atribuye al link cuando el **pago se aprueba**.

Ves el rendimiento de **cada vendedor** desde tu productora. Y cada vendedor tiene su **propio panel** para ver sus ventas.
