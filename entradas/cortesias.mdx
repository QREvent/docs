Las **cortesías** son entradas **sin cargo** que emitís vos: prensa, sponsors, staff, invitados.

## Generar un set de cortesías

Creás un **set** con un nombre para tu evento y agregás las entradas de invitado que necesites. Cada una se emite con su **código QR**, listo para validar en la puerta como cualquier entrada.

El set tiene un **código de acceso** (se genera solo, o ponés uno propio) para que el invitado pueda acceder a sus entradas.

<Note>
  Las cortesías son un canal aparte para tus invitados: **no pasan por cobro**.
</Note>
